2. Lab Scientist

3. Radiographer / Radiologist

5. HRM

Handles monitoring, dispatch, staff oversight, punctuality, accountability, operations tracking.

Now let’s go deep into this one because you asked specifically.

This is a major value point of your product.

Turnaround time means:

How long each stage of the patient/test journey takes.

Instead of just knowing “result is done,” the system knows:

when patient was registered
when case was assigned
when sample was collected
when test was started
when result was submitted
when MD reviewed it
when report was released
Why turnaround metrics matter

It helps the lab answer:

are we too slow?
which staff delays work?
which test types take too long?
which department is bottlenecked?
how long do patients wait?
who is most productive?
are urgent cases treated fast enough?

This is powerful for HRM and management.

Metrics the system should calculate
1. Registration to assignment time

How long after patient registration before work is assigned.

Formula:

assigned_at - registered_at
2. Assignment to open time

How long assigned worker takes before opening the case.

Formula:

opened_at - assigned_at
3. Open to sample collected time

How long before sample is taken.

Formula:

sample_collected_at - opened_at
4. Sample collected to started time

How long sample waits before testing starts.

Formula:

started_at - sample_collected_at
5. Started to completed time

Actual working time for the test.

Formula:

completed_at - started_at
6. Completed to submitted time

Delay after result is ready before sending to MD.

Formula:

submitted_at - completed_at
7. Submitted to review time

How long MD takes before opening/reviewing it.

Formula:

reviewed_at - submitted_at
8. Review to approval time

How long it takes to approve after review.

Formula:

approved_at - reviewed_at
9. Approval to release time

How long before patient actually gets result.

Formula:

released_at - approved_at
10. Total turnaround time

From first registration to final result release.

Formula:

released_at - registered_at
